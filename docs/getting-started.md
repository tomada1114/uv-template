# Getting Started

## Installation

```bash
pip install my-package
```

Or with [uv](https://docs.astral.sh/uv/):

```bash
uv add my-package
```

## Basic Usage

```python
from my_package import add

result = add(1, 2)
print(result)  # 3
```
